# AddColorToYourLife

A small C++ program that takes RGB values from the user and generates a solid-color PPM image.
A simple beginner-friendly introduction to pixels, color, and image formats.

## How To Run 

Compile: 
```bash  
   g++ colors.cpp -o color  
```

Run:  
```bash 
    ./color 
```
     
This Will Generate:
```bash 
 {output}.ppm
```
## Viewing the image
You can open .ppm files using: 
1.VS Code
2.GIMP / Krita
3.feh (feh output.ppm)
4.ImageMagick (display output.ppm)
